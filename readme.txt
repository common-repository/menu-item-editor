== Description ==

Manage which default WordPress menu items to show or hide. Download/install the plugin and activate it to find a new menu item named 'Menu Editor'. Within here you will be able to hide each of the default WordPress menu items.

= Docs and support =

Plugin is set to evolve but for now is pretty simple with the one interface and checkboxes
